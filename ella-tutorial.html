Ella

In this tutorial, we demonstrate how to use Emma to instrument an APK file. Note that the ella tool only narrow the coverage to the method level rather than the statement level.

The offical tutorial of ella can be found here.

Note on Ella

  1. [Instrumentation]: DO NOT REMOVE any file from the ella-out directory;
  2. [Instrumentation]: `bash ella.sh s` to start and use `bash ella.sh k` to end recording after fuzzing;;
  3. [Coverage]: `bash ella.sh s` to start and use `bash ella.sh e` to end recording after fuzzing;
  4. [Coverage]: the coverage data is named as `...dat**` is generated under one subfolder inside the ella-out directory; For example, under fold `6912ac1fa7c8d3b47cc303ef87acf2a73984c560afdeb18f8def678231d1fe73`, there will be
  5. [Error on starting server]: sudo lsof -nPi :yourPortNumber and then kill the process with port defined in ella (sudo kill -9 yourPIDnumber)
